# Table Reservation App

> A table reservation app built on ExpressJS + VueJS.

## How to use it?
```bash
# install all of the dependencies first
npm install

# to run ExpressJS server on port 5000, but you can change it in app.js
npm start

# to run VueJS
npm run serve
```
